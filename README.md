# Project_frontend_js

Одностраничное React-приложение для просмотра, поиска и фильтрации списка записей.

## Стек

- React
- Vite
- JavaScript
- CSS

## Функциональность

- Загрузка записей из публичного API
- Отображение списка карточек
- Поиск по заголовку в реальном времени
- Обработка состояния загрузки
- Обработка ошибки с кнопкой повтора
- Сообщение при пустом результате поиска
- Адаптивная вёрстка
